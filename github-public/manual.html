本ツールについて

本ツールは、複数のグラフィックが1つにまとめられた「スプライトシート画像」を、ブラウザ上で直感的に個別のパーツ画像へと切り分けるためのユーティリティツールです。 切り分けたパーツは、ZIPファイルとして一括で保存できるほか、Web制作やゲーム開発でそのまま利用できるCSSやJSON形式のコードデータとして書き出すことができます。

画像の解析・編集・保存はすべてブラウザ内で行われ、読み込んだ画像や作成したデータが外部サーバーへ送信されることはありません。GitHub Pages などで公開した場合も、通信するのはアプリ本体の静的ファイルを開くときだけです。

対応環境：マウスとキーボードを使った細かな編集を想定したPC向けツールです。横幅 1280px 以上の画面での利用を推奨します。

基本的な使い方の流れ

  1. 画像の読み込み
    画面右上の「画像を読み込む」ボタンをクリックします。初めてなら「サンプルで試す」で、内蔵の8個のスプライトをすぐに確認できます。「サンプル2：背景透過」は緑背景のドラゴンを読み込み、緑色を指定済みの透過画面を開きます。「透過を適用する」を押すと、そのまま自動検出まで試せます。PNG / JPG / WebP / GIF と、このアプリが出力したZIPを読み込めます。
  2. 範囲選択（スライス）
    左パネルから適したモードを選択して、画像の切り出し枠（スライス）を作成します。
  3. スライスの調整と命名
    枠線をドラッグしてサイズや位置を微調整し、名前を設定します。
  4. 保存・書き出し
    画面右下の「すべてをZIPで保存」ボタンを押します。編集済みスプライトシート、個別PNG、JSON、CSSが1つのZIPにまとまり、ZIPを読み込めば作業を復元できます。

高機能背景透過機能

画像内の特定の背景色（単色または類似色）を検出し、高度なピクセル制御アルゴリズムによって非常に綺麗に透過・除去することができます。外部サーバーを介さずブラウザ上で安全に動作します。

処理上限：背景透過は 800万画素までです。大きい画像は縮小してから処理してください。

  • 透過する基準色の選択： モーダル内のスポイトツールを使用し、プレビュー画像上の背景色をクリックして透過させたい色を直接選択できます。カラーピッカーで直接調整することも可能です。
  • 色の許容値： 基準色に対してどの程度類似した色まで透過するかを調整します。輪郭付近のアンチエイリアス（ぼかし）やグラデーションも滑らかに処理します。
  • 境界のぼかし (Feather)： 切り抜き境界を半透明でなだらかにぼかし、エッジのギザギザ（ジャギー）を無くします。
  • 境界の収縮 (Choke)： 透過境界を内側に数ピクセル狭めることで、エッジにわずかに残ってしまった元の背景色の輪郭を完全にカットします。
  • フリンジ除去 (De-fringe)： 半透明のエッジ部分に元の背景色（白い背景なら白っぽいにじみ）が残ってしまう現象を防ぐため、エッジのピクセルカラーを隣接する不透明（被写体側）なピクセルの色で自動的に補正します。別の背景に置いたときに美しく馴染みます。
  • 適用範囲の選択： 「画像全体」の透過のほか、「選択中のスライス範囲のみ」にも透過を適用できます。特定のキャラクターやパーツのみ背景を除去したい場合に便利です。

リアルタイムプレビュー対応： スライダーなどの値を変更すると、モーダル内のプレビュー画面にリアルタイムで仕上がりが反映されます。内容を確認し、問題なければ「透過を適用する」ボタンでキャンバスに反映します。この処理も元に戻す（Undo）ことができます。

サンプル画像2（ドラゴン）： 緑背景の画像で背景透過を体験するための内蔵サンプルです。「サンプル2：背景透過」を押すと、基準色 #00FF00 と推奨値を設定済みの状態で開きます。プレビューを確認して「透過を適用する」を押すと、透明化後にスライスを自動検出します。

2つの分割モード

1. 自動検出モード

画像内の透明（アルファ値）を自動で解析し、グラフィックが描画されている領域を自動で囲んでスライス枠を作成します。

  • 透明度の閾値： どの程度透明に近いピクセルを「透明な背景」と見なすかを設定します（値が大きいほど、薄い半透明のピクセルも背景として除外されます）。
  • 最小サイズ： これより小さいドットの塊（ゴミや細かなハイライトなど）を検出対象外にします。
  • 処理上限： 安定動作のため、自動検出は800万画素・2,000領域までです。大きい画像や大量の領域は手動範囲選択を使ってください。

2. 手動範囲選択モード

画像上を直接マウスでドラッグして、切り分けたい領域を自由に作成します。不規則な配置のシートを手動で個別に切り出したい場合に適しています。

細かな微調整機能とショートカット

枠のサイズ変更・移動

作成されたスライス枠は、クリックして選択することができます。 選択された枠の四隅のポインターをドラッグするとサイズを変更でき、枠の中をドラッグすると位置を移動できます。

便利なショートカットキー

  • Cmd / Ctrl + C ： 選択中のスライスをコピー
  • Cmd / Ctrl + V ： コピーしたスライスを貼り付け
  • Cmd / Ctrl + Z ： 直前の操作を取り消す（元に戻す）
  • Cmd / Ctrl + Y ： 取り消した操作をやり直す
  • Delete / Backspace ： 選択中のスライスを削除
  • 矢印キー (↑/↓/←/→) ： 隣接するスライスへ選択を移動

画像編集（移動・回転・リサイズ・消しゴム）

  • 画像編集モード： 右パネルの「画像編集」ボタンを有効にすると、選択した枠内の「画像データ自体」を操作できます。
    • 移動： 枠の中をドラッグすると、画像ピクセルを枠ごと移動させます。
    • 回転： 枠の上部に表示される「回転ハンドル（青い丸ポインター）」をドラッグするか、右パネルの「回転角」コントロールを操作して自由に回転できます。
    • リサイズ： 枠の四隅のアンカーをドラッグすると、枠のサイズ変更に合わせて中の画像も拡大・縮小して適用されます。
    いずれの操作もドラッグを離した時点で画像ピクセルが背景画像に上書き適用されます。
  • 消しゴムツール： スライス内の不要なドットをなぞって透明に消去することができます。

データの保存と管理

一時保存（セーブスロット）

現在の作業状態（読み込んだ画像、作成したスライス枠すべて）をブラウザ内に最大20スロットまで一時的に保存できます。 ブラウザのIndexedDBを利用するため、通常はブラウザを閉じてもデータは保持されます。IndexedDBを利用できない環境では、現在開いている間だけのメモリ保存に自動で切り替わります。作業を一時中断して後から再開したい場合にご活用ください。

共有PCでの注意：一時保存には画像そのものが含まれます。作業後は左側の「すべての一時保存を削除」で保存スロットを消去してください。機微な画像は一時保存せず、専用のブラウザプロフィールまたは専用ドメインで利用することをおすすめします。

ZIP書き出しの上限：安定して書き出せるよう、切り出す画像の合計は 1,600万画素までです。大量・高解像度のスライスは、作業を分けて保存してください。
